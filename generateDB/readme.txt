- Scriptul SQL si programul Java GetTheJSONS se folosesc pentru crearea si popularea bazei de date
Pasi:
	1. Intrati in mysql workbench, stergeti versiunea bazei de date pe care o aveti, si executati noobs_script.sql
	2. Deschideti in IntelliJ proiectul GetTheJSONS, si in clasa Main, in metoda addToDB, modificati parametrii
bazei de date cu userul si parola voastra
	3. Executati programul
	HALLELUJA!
